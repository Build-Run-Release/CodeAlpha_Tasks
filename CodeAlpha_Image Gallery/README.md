# AI Visionary Image Gallery

This is a responsive, interactive image gallery showcasing AI-generated artwork. It features a modern design with category filters and a lightbox viewer.

## Features

*   **Responsive Grid Layout**: Adapts seamlessly to different screen sizes.
*   **Topic Filtering**: Filter images by category: Nature, Abstract, Architecture, and Technology.
*   **Lightbox Modal**: Click on any image to view it in full screen with a dark overlay.
*   **Modern Aesthetics**: Premium dark theme with vibrant accents and smooth animations.

## How It's Built

*   **HTML5**: Semantic structure with `data-*` attributes for filtering logic.
*   **CSS3**: Custom properties (variables), Flexbox, Grid, and CSS transitions/animations. No external frameworks were used.
*   **JavaScript**: Plain Vanilla JS for handling filter events and the lightbox functionality.

## Setup & Usage

1.  Clone the repository or download the folder.
2.  Open the `index.html` file in any modern web browser.
3.  Interact with the filter buttons to sort images.
4.  Click on an image to open the lightbox.
