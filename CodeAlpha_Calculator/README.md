# Basic Calculator

This is a simple, functional calculator built with HTML, CSS, and JavaScript.

## Features
- Basic arithmetic operations: Addition, Subtraction, Multiplication, Division
- Clear button to reset calculations
- Responsive design
- Keyboard input support (via button clicks)

## How It Works
- **index.html**: Contains the structure of the calculator.
- **style.css**: definitions for the visual styling of the calculator, using CSS Grid for layout and improved aesthetics.
- **script.js**: Contains the JavaScript logic for handling button clicks and computations.

## How to Run
Simply open the `index.html` file in any modern web browser.

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
